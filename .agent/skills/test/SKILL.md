# Test Skill

## When to use this skill

- Use this when you want to test something
- This is helpful for testing

## How to use it

Since my app uses a different python virtual environment, you need to use the follwing path to run it:

```bash
~/Documents/virtual-env/makix-qwen-tts/bin/python
```    

You can run the app with the following command:
```bash
~/Documents/virtual-env/makix-qwen-tts/bin/python app.py
```

For installing dependencies, use the following command:

```bash
~/Documents/virtual-env/makix-qwen-tts/bin/pip install -r requirements.txt
```
